**Installation**
--------------

To run the projects locally, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/dakshbhatnagar/projects.git
   cd projects
   ```

2. **Install the required Python libraries:**

   ```bash
   pip install -r requirements.txt
   ```
